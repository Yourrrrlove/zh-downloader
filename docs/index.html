一个非常给力的用于嗅探和下载知乎视频的浏览器插件

Download from Google Store Download from Github

功能截图

全自动嗅探知乎视频资源, 并提供丰富多彩的功能:比如下载/复制/分享等

功能特性

现代化的知乎视频下载器，采用最新技术栈，功能强大且易于使用

支持 Manifest V3

安全、现代、兼容 Chrome 最新扩展平台，确保长期稳定运行

极简 UI

采用 React + TailwindCSS，界面科技感十足，现代化设计体验

一键下载

支持 mp4/ogg 等主流格式视频的直接下载，操作简单快捷

任意帧截图

可对当前视频任意时间点进行截图保存，精准捕捉每一个瞬间

历史记录

自动保存下载与截图历史，方便回溯和管理已下载内容

多主题支持

亮色/暗色主题自由切换，适应不同使用环境和个人喜好

视频滤镜

支持亮度、对比度、灰度、模糊、色相等多种滤镜效果调节

画中画模式

支持 PIP 画中画功能，可在其他应用上方悬浮播放视频

倍速播放

支持 0.25x 到 3x 倍速调节，满足不同观看需求

视频变换

支持水平翻转、垂直翻转、旋转等视频变换效果
